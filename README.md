# Web An toàn

Mới có đăng nhập dùng Node.js, Express, Passport, Mongoose, EJS...

### Version: 2.0.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
# Chạy được luôn không cần cấu hình gì cả
```

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
